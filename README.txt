Этот проект - вторая версия онлайн игры крестики-нолики.

Что отличается от первой версии:
1) Сервер игры стал отказоустойчивым. Для этого запускается несколько серверов, и в случает отказа первого сервера,
 все идущие сеансы игры переносятся на другой сервер.
2) Проведен рефакторин кода.
3) Изменилась структура отправляемых сообщений.


Для соединения игроков с сервером и между собой используются сокеты.
В игре поддерживается чат.
Интерфейс реализован с использованием библиотеки gtk 2.0
Для компиляции есть Makefile.
Игра запускалась только компьютерах с Ubuntu.
